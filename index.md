## üõ† Overview
The Gesture Hardware Controller allows you to control physical devices, such as LED lights and motors, using your hands. ### How It Works
1. **Hand Tracking:** The system uses MediaPipe to track your hand movements in real-time.
2. **Gesture Recognition:** The software in Python recognizes what gestures you make.
3. **Serial Communication:** It communicates with Arduino hardware to control devices.
4. **Device Control:** Use gestures to control motors and lights with ease.
